# Holiganbet Portal

Holiganbet erişim ve bilgilendirme için hazırlanmış modern web arayüzü.

## Hakkında

Bu proje, Holiganbet kullanıcıları için hızlı erişim ve güncel bağlantı yönlendirme amacıyla oluşturulmuştur. Mobil uyumlu yapısı sayesinde tüm cihazlarda sorunsuz çalışır.

## Özellikler

- Hızlı yönlendirme sistemi
- Mobil uyumlu tasarım
- SEO dostu yapı
- Basit ve hafif frontend
